# AURA INTIMATES — Luxury Innerwear Ecommerce

A premium luxury lingerie & innerwear ecommerce site built with React + Vite.

## Tech Stack
- **React 18** + **Vite**
- **Framer Motion** — cinematic page transitions & animations
- **React Router DOM** — client-side routing
- **Tailwind CSS v3** — utility-first styling with custom luxury palette
- **Headless UI** — accessible Cart sidebar & Modals
- **React Hot Toast** — elegant notifications

## Getting Started
```bash
npm install
npm run dev       # dev server at http://localhost:5173
npm run build     # production build
```

## Payments
- Checkout now uses a fake in-app payment prompt for testing M-PESA and Visa flows.
- M-PESA accepts any 4-digit PIN in the mock modal.
- Visa accepts any valid-looking card details in the mock modal.

## Key Features
- 22 sample products priced in KES
- Cart with localStorage persistence + free shipping progress bar
- Wishlist with localStorage persistence
- Product filtering (color, size, price range) + sort options
- Quick view product modals
- Image hover-swap on product cards
- Page transition animations (Framer Motion)
- Scroll reveal animations
- Fully responsive mobile-first design
- Payment placeholders: M-Pesa, Visa
